# Job Hunt Dashboard

Automated job application pipeline with visual kanban board.

## Features

- **7-Stage Pipeline**: New → Qualified → Applying → Applied → Interview → Offer → Closed
- **Drag-and-Drop**: Move jobs between stages
- **AI Scoring**: Automatic job matching and scoring
- **Resume Tailoring**: Auto-generated custom resumes per job
- **Application Tracking**: Full pipeline visibility

## Deploy on Render

1. Create new Web Service
2. Connect GitHub repo
3. Set environment:
   - Build Command: `npm install`
   - Start Command: `npm start`
4. Deploy

## Local Development

```bash
npm install
npm run dev
```

Open http://localhost:3000
